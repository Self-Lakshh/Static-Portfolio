## ðŸ“¦ Getting Started

1. **Clone the repository:**
   ```sh

   cd static-portfolio
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm run dev
   ```
4. **Open in your browser:**
   Visit `http://localhost:5173` (or the port shown in your terminal).
